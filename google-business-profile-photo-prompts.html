Universal photo workflow

Core GBP shot list template

Build a simple 5-photo set first:
1. Exterior or arrival shot so buyers recognize the location or branded vehicle.
2. Team-in-action photo that shows real service delivery.
3. Before-and-after or process proof image.
4. Product, treatment-room, tool, or equipment close-up.
5. Trust-building owner or staff photo.

Rule: each photo should answer one buyer question — "What do they do?", "Can I trust them?", or "What happens if I hire them?"
  • Best for businesses that need a fast first-pass image plan before getting niche-specific.
  • Use this when the current profile feels generic, outdated, or full of random uploads.
Captions + context

Simple photo caption prompt

Caption formula:
[Service] in [City/Area] + [specific proof detail] + [light CTA].

Example:
Water-heater replacement in Mesa with same-day install and haul-away completed for a local homeowner. Call to schedule an estimate.

Use short, literal wording. The photo carries the emotion; the caption should carry clarity.
  • Useful when the business uploads photos but gives Google and buyers no context.
  • Pairs well with the GBP post prompt library for weekly content reuse.
